# Manual-VS-Code-Server-Installer-for-WSL-Remote-SSH
This repository provides a **manual installer for VS Code Server** in WSL or remote Linux environments.   It is designed to bypass download issues caused by proxies, firewalls, or Remote-SSH plugin limitations.

## Features

- Works for WSL or any Linux remote host.
- Uses `wget` with proxy support if configured in your environment.
- Avoids VS Code Remote-SSH download locks.
- Compatible with `x64` Ubuntu or Debian-based systems.
- Can be reused for upgrading VS Code Server manually.
